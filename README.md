# Как Изменять переменные?

hui Никак незменяемая
piz название-переменно = значение
construct обращение к переменной такое название-переменной.Параметр-там = значение
const можно менять как обычные перемменные НО только тогда когда ты написал таким вариантом const piz // Дальше переменная
